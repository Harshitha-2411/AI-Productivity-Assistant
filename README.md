# AI Productivity Assistant

AI productivity assistant that manages daily tasks and planning using Python and Llama 3.

## Features

- AI assistant for productivity
- Add tasks
- View tasks
- Complete tasks
- Delete tasks
- AI daily planning
- Task dashboard

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3
- SQLite

## Project Structure

AI-Productivity-Assistant  
│  
├── app.py (Streamlit interface)  
├── ai_engine.py (AI assistant logic)  
├── tasks.py (task management)  
├── database.py (database connection)  
├── tasks.db (SQLite database)  
└── data/
